# Task 2 - Data Cleaning

This project performs basic data cleaning on the Students Performance dataset using Python and Pandas.

## Objective
To clean and prepare the dataset by handling duplicates and missing values.

## Steps Performed
- Loaded dataset using Pandas
- Removed duplicate rows
- Filled missing values with default values
- Displayed cleaned dataset
- Checked missing values after cleaning

## Technologies Used
- Python
- Pandas

## Output
Cleaned dataset without duplicates and missing values issues.
